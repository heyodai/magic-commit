## Development

### Pre-requisites
- Python 3.6+
- Poetry

### Poetry Commands
- `poetry install` - Install dependencies
- `poetry run git-diff` - Acquire git diffs via API
- `poetry run upload` - Upload contents of `/data` to AWS
- `poetry run download` - Download contents of `/data` from AWS
